Step 1

Install the Extension

BambiBrowser+ is a developer-mode extension — not published in any store. It takes about two minutes to load. Choose your browser below.

chrome://extensions/
1
Open the extensions page

2
Enable Developer Mode
Look in the top-right corner for the Developer mode label.

3
Click "Load unpacked"

4
Select the extension folder

5
Pin the icon to your toolbar
Click the puzzle-piece 🧩 icon in the toolbar, find BambiBrowser+, and click the pin icon so the 🐰 icon always shows.

Step 2

Start Bambi Player

The extension needs a small background helper running on your PC. It receives video URLs from the browser and launches VLC. Without it, the browser falls back to normal in-page playback.

Standard
Single monitor
1
Open the BambiBrowser+ folder
Navigate to wherever you extracted or cloned the files.
2
Double-click bambi_player.exe
No window opens — the player runs silently in the background.
3
Check the system tray
A 🐰 icon appears in the tray (bottom-right of the taskbar). That means the helper is awake and listening.
BambiBrowser+
✓To stop the player later, right-click its tray icon and choose Quit.
Multi-monitor mode
2–3 screens at once
1
Open PowerShell in the BambiBrowser+ folder
Right-click inside the folder → Open in Terminal.
2
Run the multi-monitor script
.\start_bambi_player_multimon.ps1
3
Check the tray
Same 🐰 icon, same helper. The extension doesn't need to know which mode you're running.
🖥Monitor 1 plays with full audio. Additional monitors mirror the video muted, all fullscreen.
⚠VLC must be installed and reachable. The player helper looks for VLC in the bundled vlc/ subfolder first, then falls back to a system-wide VLC installation.
Step 3

Start on Boot (optional)

Skip the manual double-click every session — have the player start automatically when you log in to Windows.

Option A
Startup Folder

Easiest method. Anything placed in the Windows startup folder launches when you log in.

1
Open the Startup folder
Press Win + R, type shell:startup, press Enter.
2
Create a shortcut
Right-click bambi_player.exe → Create shortcut. Drag or copy that shortcut into the startup folder.
3
Done
Next time you log in, the tray icon will appear automatically within a few seconds.
Option B
Task Scheduler

Cleaner — no UAC flash, starts silently at login.

1
Open Task Scheduler
Press Win, search Task Scheduler, open it.
2
Create Basic Task
In the right panel click Create Basic Task… — name it e.g. BambiPlayer.
3
Set trigger: When I log on
On the Trigger screen choose When I log on. Click Next.
4
Set action: Start a program
Choose Start a program, then Browse and navigate to bambi_player.exe. Click Next → Finish.
ℹYou can always flip Bambi Mode off from the popup without quitting the player — the player just sits idle until you re-enable it.

General tab
Master controls & playback settings
Bambi Mode section
Enable hijack
The master on/off switch. When ON, the extension actively watches matched domains and hijacks video to VLC. When OFF, the extension is completely dormant. Toggling it ON while on a matched tab triggers the hijack immediately without reloading.
Playback section
Multi-monitor
When ON, the video URL is sent to the multi-monitor helper, which opens a fullscreen VLC window on every connected display simultaneously. Monitor 1 plays with full audio; all others are muted. Keyboard and pointer lock are always engaged in this mode. Requires the multi-monitor player script.
Playback section
Manual mode click sensitivity
Controls how tightly a user click must be paired with a video play event on MANUAL domains.

Strict — ~1.2 s window. Balanced — 2.2 s (default). Relaxed — 3.2 s, more forgiving for slow-loading embeds.

Domain modes
AUTO vs MANUAL
hypnotube.com AUTO
example.com MANUAL
AUTO — the extension proactively scans for a video element when the page loads and hijacks it automatically.

MANUAL — the extension waits for you to click play, intercepts that event, pauses the browser player, and hands the URL to VLC.
Adding a domain
Type the domain → Add
Type any domain (e.g. somesite.com) and click Add or press Enter. Custom domains get the MANUAL badge. www. is stripped automatically.
⚙ Configure (manual domains only)
Player learning mode
Click the ⚙ cog next to a MANUAL domain. A crosshair overlay appears — click the main video player on the page once. The extension remembers that element for accurate future hijacks.
Manual tab
Keyboard shortcut & local file playback
Manual trigger section
Enable shortcut trigger
When enabled, pressing the configured keyboard shortcut on any page activates a targeting overlay. Click any video and it is sent to VLC — bypassing domain matching entirely. Useful for one-off sites.
Keyboard shortcut picker
Default: Alt+Shift+V
Four options: Alt+Shift+V, Ctrl+Shift+V, Alt+V, Ctrl+Alt+V. Pick whichever doesn't conflict with your sites.
Local file section
Browse → Play
Pick any video file from your local disk and send it directly to VLC without needing a URL. Click Browse, select your file, then click ▶ Play.
Blacklist tab
Skip specific URLs on watched domains
How it works
URL substring matching
Each entry is checked against the full URL of the current page. If the URL contains the entry anywhere (case-insensitive), the extension skips that page — even if the domain is active.
Examples
Useful patterns
preview→ skips any URL containing "preview"
category/shorts→ skips that specific path
?embed=→ skips pages with an embed query param
Adding & removing
Type the pattern → Add
Type any keyword or path segment and click Add. Entries appear in the list — click ✕ to remove. The protocol (https://) is stripped automatically.
When you browse
AUTO domains
What happens on e.g. hypnotube.com
1
Page loads
The extension checks domain match, blacklist, and ignored-paths list. If any check fails, the page is left alone.
2
Video element found
The extension scans for the main <video> element with a valid source URL — it won't fire on ad pre-rolls.
3
If the player is running
Browser video is paused. The URL is sent to bambi_player. VLC opens fullscreen. Pointer and keyboard lock activate.
4
When the video ends
VLC closes, locks release, and the page returns to normal. The extension resets and is ready for the next video.
5
If the player is NOT running
The extension falls back gracefully — it unmutes and autoplays the video in the browser as normal.
MANUAL domains
What happens on custom-added sites
1
Page loads — extension waits
Nothing happens automatically. The extension watches in the background but does not act until you do.
2
You click play
The extension detects your click, identifies which video element is closest, and waits for its play event.
3
Hijack fires within the sensitivity window
If the play event arrives within the configured window (default 2.2 s), the extension pauses the browser player and sends the URL to VLC.
4
Rest is the same as AUTO mode
VLC opens fullscreen, locks activate, everything resets when the video ends.
💡If a site loads a long pre-roll ad before the main content, the extension waits an 18-second grace period on eligible domains before considering a play event as intentional.
v5.2 Feature NEW

Input Lock Timer

In multi-monitor mode, keyboard and pointer locks are always active while a video plays. In single-monitor mode these locks are optional — and you can start a lockdown timer that prevents turning input lock off for a chosen duration.

New in 5.2
Input lock (single-monitor)
General tab → Playback section
Input lock (single-monitor)
When enabled, the keyboard and pointer lock engage at the start of a VLC fullscreen session — exactly as they do in multi-monitor mode — but only for single-monitor setups.

When disabled (default), single-monitor sessions play without any input lock. Multi-monitor is unaffected — it always locks regardless of this toggle.
General tab → Playback section
Lockdown duration
Lockdown duration 1 hour ▾
Only visible when Input lock is enabled. Choose how long lockdown should prevent disabling input lock:

10 minutes · 20 minutes · 1 hour (default) · 2 hours · 4 hours · 6 hours · 8 hours · 12 hours

Then click Start Lockdown. During lockdown, input lock cannot be disabled from settings. When the timer expires, input lock is not turned off automatically — the timer only unlocks the setting so you can disable it yourself if you want.
ℹ Set the timer to match your typical session length. The timer resets each time a new video starts fullscreen — it doesn't carry over between videos.
⚠ While input lock is active, keyboard shortcuts and mouse movement are suppressed in the browser. VLC still receives input normally through its own window. In multi-monitor mode, input lock is always forced on. In single-monitor mode, lockdown only controls whether the setting can be switched off.
v5.2 Feature NEW

Auto-Play Fallback

Set a URL that automatically opens in a new browser tab if Bambi Mode is active but no video has been detected for a configurable amount of idle time. Keeps sessions going without you needing to navigate manually.

New in 5.2
Auto-Play Fallback
General tab → Auto-Play Fallback section
Open URL if no video found
Master toggle for the auto-play fallback. When enabled and Bambi Mode is active, a countdown begins as soon as the extension detects an idle state (no video sent to VLC yet). When the countdown expires, the configured URL opens in a new tab.

The timer automatically resets when:
• You navigate to a new page on a matched domain
• A video is successfully sent to VLC (timer cancels permanently for that session)
• You disable the toggle

After the URL opens, the timer reschedules itself so consecutive idle periods keep triggering the fallback until a video is found.
Fallback URL
Enter the URL to open
Paste any https:// URL here — typically a page on a site you've already added to your active domains. When the idle timer fires, this URL opens in a new browser tab.
Open after idle for
How long to wait before opening the URL
Open after idle for 10 minutes ▾
Choose the idle window that suits your browsing style:

5 minutes · 10 minutes (default) · 15 minutes · 20 minutes · 30 minutes · 45 minutes · 1 hour

Shorter times are useful if you tend to leave the browser idle and want it to pick up content quickly. Longer times leave more room for you to browse and click play yourself before the fallback kicks in.
✓ The fallback URL only opens when Bambi Mode is active and no video has been sent to VLC during the current idle window. It won't interrupt an active VLC session.
ℹ The fallback URL should be a page on a site already in your active domains list — otherwise the extension won't try to hijack its video automatically when it loads.
