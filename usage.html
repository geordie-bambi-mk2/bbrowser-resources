Using the Popup

Main controls are under General, Domains, Manual, and Blacklist tabs.

General tab

  • Enable hijack master toggle.
  • Multi-monitor toggle.
  • Input lock toggle + lockdown timer controls.
  • Auto-play fallback URL controls.

Domains, Manual, Blacklist

  • Domains: add watched domains, preset one-click adds, player learning for manual sites.
  • Manual: shortcut trigger and local file playback to VLC.
  • Blacklist: URL substrings that suppress hijack.
