Start On Boot

Automatically run the helper every login so the extension is always ready without manual startup.

Version v6.2 · Step 3 of 4

Startup Folder Method

Fastest no-admin setup for most users.

Step 1

Open Run

Press Win + R.

Step 2

Open Folder

Type shell:startup and press Enter.

Step 3

Add Shortcut

Create a shortcut to bambi_player.exe.

Step 4

Confirm

Log out and back in. Popup should show helper connected.

Task Scheduler Method

Best when you want explicit launch control.

Task Setup

Trigger + Action

Create a basic task, set trigger to When I log on, and action to Start a program targeting bambi_player.exe.

Validation

Manual Test Run

Right-click the task and choose Run. Check tray icon and extension helper status immediately.

If helper status is connected after login, boot setup is complete.
