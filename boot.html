Start On Boot (Optional)

Automatically run the helper every login so the extension is always ready without manual startup.

Version v6.1
Boot Options
  • Manual Start
  • Auto-start Methods
  • Check Status

Recommended Methods

Option A: Startup Folder

  1. Press Win + R.
  2. Run shell:startup.
  3. Create shortcut to bambi_player.exe.
  4. Place shortcut in startup folder.

Option B: Task Scheduler

  1. Create a basic task.
  2. Trigger: When I log on.
  3. Action: Start a program.
  4. Select bambi_player.exe.

Validation

After reboot/login, open the extension popup. If helper status is connected, auto-start worked correctly.
