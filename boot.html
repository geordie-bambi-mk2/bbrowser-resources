Start on Boot (Optional)

Pick whichever method you prefer so the helper starts automatically at login.

Option A: Startup folder

  • Press Win + R and run shell:startup.
  • Create a shortcut to bambi_player.exe.
  • Place the shortcut in the startup folder.

Option B: Task Scheduler

  • Create basic task, trigger: When I log on.
  • Action: Start a program.
  • Select bambi_player.exe and finish.
