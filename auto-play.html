Auto-Play Fallback

If Bambi Mode is active and no video is sent to VLC before idle timeout, the extension can open a fallback URL automatically.

Version v6.1
Fallback Flow
  • Setup
  • General Usage
  • Advanced Notes

Configuration

How To Enable

  1. Turn on Open URL if no video found.
  2. Enter fallback URL.
  3. Choose idle delay window.

Recommended URL

  • Use a URL on an allowed domain.
  • Avoid static pages with no playable videos.
  • Prefer pages where your normal flow starts.

Runtime Behavior

  • Timer runs while Bambi Mode is enabled and page remains idle for playback.
  • When timeout hits with no VLC send, fallback URL opens in a new tab.
  • Successful VLC send cancels fallback for that active session.
