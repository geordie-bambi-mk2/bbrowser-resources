Auto-Play Fallback

If Bambi Mode is active and no video is sent to VLC before idle timeout, the extension can open a fallback URL automatically.

Version v6.2

Configuration Steps

Set once, then fallback runs automatically during idle sessions.

Step 1

Enable Fallback

Turn on Open URL if no video found in settings.

Step 2

Choose URL

Provide a fallback URL that has reliable playable video content.

Step 3

Set Delay

Pick idle delay long enough to avoid unwanted tab openings.

Step 4

Validate

Run a short idle test and confirm the fallback tab opens as expected.

Runtime Behavior

Timer

Idle Countdown

Countdown runs while Bambi Mode is active and no successful VLC send occurs.

Suppress

Active Playback Guard

When helper reports likely active playback, fallback is deferred until safe.

Trigger

Open New Tab

On timeout with no send, fallback URL is opened by the extension runtime.

Cancel

Successful Send

When VLC send succeeds, fallback is cancelled for the current session context.

Use fallback URLs on allowed domains whenever possible to keep behavior predictable.
