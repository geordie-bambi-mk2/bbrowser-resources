Auto-Play Fallback

If Bambi Mode is active and no video is detected in time, open a fallback URL automatically.

Setup

  1. Enable Open URL if no video found.
  2. Enter a fallback URL.
  3. Select idle delay.

Behavior

  • Timer runs only while Bambi Mode is active.
  • If no video is sent to VLC before timer expires, URL opens in a new tab.
  • On successful VLC send, fallback timer is canceled for that page session.
