Input Lock

Input lock is independent from multi-monitor. You can enable or disable it regardless of monitor mode.

Version v6.1
Input Lock Flow
  • Rules
  • Where To Configure
  • Technical Notes

Behavior Rules

  • Input lock ON: keyboard/pointer restrictions apply during VLC fullscreen sessions.
  • Input lock OFF: playback still works, but no input restriction is enforced.
  • Multi-monitor and input lock are separate controls.
  • During lockdown, input lock cannot be toggled off.

Lockdown Setup

  1. Enable Input lock.
  2. Select lockdown duration.
  3. Press Start Lockdown.

After Timer Ends

  • Lockdown restriction is removed.
  • Input lock state stays ON until manually changed.
  • You regain the ability to disable it.
