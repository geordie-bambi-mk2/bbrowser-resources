Input Lock

Input lock is separate from multi-monitor mode and now works without admin rights using low-level keyboard and mouse hooks.

Version v6.2

Behavior Rules

When On

Keyboard + Mouse Block

Input events are swallowed during active VLC lock sessions.

When Off

Playback Only

VLC playback still launches, but input remains unrestricted.

Independent

Separate Toggle

Input lock can be managed regardless of multi-monitor setting.

Lockdown

Timed Enforcement

During lockdown, users cannot toggle lock off until timer expiry.

System security shortcuts like Ctrl+Alt+Del are OS-level and cannot be blocked by user-space applications.

Lockdown Flow

Setup

Start Lockdown

Enable input lock, choose a duration, and start lockdown from settings.

After Expiry

Toggle Restored

Lockdown enforcement ends and user control returns while retaining current lock state.
