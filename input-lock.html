Input Lock v5.2

Single-monitor can opt in; multi-monitor always forces input lock ON.

Rules

  • Multi-monitor ON: input lock is forced ON.
  • Multi-monitor OFF: you may toggle input lock on/off.
  • During lockdown, input lock setting cannot be disabled.
  • When lockdown timer ends, input lock remains ON; only the restriction is removed.

How to use lockdown

  1. Enable Input lock (or enable multi-monitor).
  2. Select Lockdown duration.
  3. Click Start Lockdown.
