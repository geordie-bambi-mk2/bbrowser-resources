Start Bambi Player Helper

The helper listens on http://127.0.0.1:5655, launches VLC, and streams live playback status back to the extension.

Version v6.2 · Step 2 of 4

Starting The Helper

The helper runs as a tray application with no visible window.

Launch

Run The EXE

Double-click bambi_player.exe. A tray icon appears in the system notification area.

Verify

Check Popup

Open the browser extension popup. Status should show Helper Connected in green.

Stop

Exit via Tray

Right-click the tray icon and select Exit to stop the helper cleanly.

Logs

bambi_player.log

Log file sits alongside the EXE. Check it if the helper fails to start or VLC doesn't launch.

Live Playback Status

The popup reflects the helper's live state while VLC is running.

Progress

Timeline Display

Current position, total duration, and remaining time update live. A progress bar reflects the timeline fraction reported by the RC interface.

State

Session Info

Popup shows how many VLC instances launched and whether multi-monitor mode is active for the current session.

Multi-Monitor Behavior

Controlled by the toggle in the extension popup.

Off

Primary Only

VLC launches fullscreen on the primary monitor only. Other displays are left untouched.

On

All Monitors

VLC launches fullscreen on every detected display simultaneously with the same content.

Audio

Primary Only

Only the primary monitor instance carries audio. Mirror displays are muted.

RC

Status Channel

The primary VLC instance exposes a local RC port so the helper can query playback position and state.
