Start Bambi Player

The extension sends URLs to the helper running on http://127.0.0.1:5655.

Single monitor

  • Run bambi_player.exe.
  • Confirm the tray icon appears.
  • When online, popup status dot is green.

Multi-monitor

  • Run .\start_bambi_player_multimon.ps1.
  • Primary display carries audio.
  • Other displays mirror video muted.
