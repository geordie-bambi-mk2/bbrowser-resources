Start Bambi Player Helper

The helper listens on http://127.0.0.1:5655, launches VLC, and streams live playback status back to the extension.

Version v6.1
Player Focus
  • Install
  • Start Helper
  • Popup Status
  • Advanced Details

What To Start

Run The EXE

  • Launch bambi_player.exe.
  • Confirm tray icon appears.
  • Popup should show Helper Connected.

Playback Status

  • When VLC is active, popup shows playing status.
  • Current / total / remaining time updates live.
  • Progress bar reflects timeline position.

Multi-monitor Behavior

  • OFF: Primary monitor only.
  • ON: All detected monitors.
  • Primary monitor carries audio, mirror displays are muted.
