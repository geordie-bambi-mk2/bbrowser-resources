Install The Extension

Set up BambiBrowser+ on any Chromium browser in under two minutes, then open the full settings tab from the popup cog.

Version v6.1
Setup Flow
  • 1. Load Extension
  • 2. Start Helper
  • 3. Auto-start
  • 4. Daily Usage

Install Steps

Browser Setup

  1. Open chrome://extensions/ (or equivalent).
  2. Enable Developer mode.
  3. Click Load unpacked.
  4. Select this repo's extension folder.

Post-install Checks

  • Pin BambiBrowser+ to the toolbar.
  • Open popup and confirm the compact status view appears.
  • Click the cog to open full-size settings in a tab.

Notes

Unpacked extension warnings are expected in developer mode. They do not affect local functionality.
