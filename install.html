Install The Extension

Set up BambiBrowser+ on any Chromium browser in under two minutes, then open the full settings tab from the popup cog.

Version v6.2 · Step 1 of 4

Browser Setup

Load the unpacked extension into any Chromium-based browser.

Need the project files first?
Direct download of the latest BambiBrowser_PlusPlus source archive from GitHub (main branch).
Download main.zip
Step 1

Open Extensions

Navigate to chrome://extensions/ or your browser's equivalent extensions page.

Step 2

Developer Mode

Toggle Developer mode on using the switch in the top-right corner of the page.

Step 3

Load Unpacked

Click Load unpacked and select this repo's extension/ folder.

Step 4

Pin & Verify

Pin the icon to the toolbar. Open the popup and confirm the compact status view appears.

Post-Install Checks

Confirm the extension loaded correctly before moving on.

Popup

Status View Loads

Clicking the icon should show the compact status popup — helper state, monitor mode, input lock, and a playback section.

Settings

Cog Opens Full Tab

Clicking the cog icon in the popup should open a full-size settings tab, not a cramped popup panel.

Unpacked extension warnings in the browser toolbar are expected in developer mode. They do not affect local functionality.
