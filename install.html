Install the Extension

Quick setup for any Chromium-based browser (Chrome, Edge, Brave, Opera).

Steps

  1. Open your extensions page (for Chrome use chrome://extensions/).
  2. Enable Developer Mode.
  3. Click Load unpacked.
  4. Select the extension folder in this project.
  5. Pin BambiBrowser+ to your toolbar.

Tip

If the browser warns about unpacked/developer extensions, you can safely dismiss it for this local setup.
