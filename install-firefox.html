Install on Firefox

BambiBrowser+ for Firefox is distributed as a signed .xpi file. Download it from the GitHub repo and install in one step — no developer mode needed.

Version v6.2 · Firefox 140+

Download & Install

The extension is signed by Mozilla so regular Firefox accepts it permanently.

Step 1

Get the .xpi file

The signed .xpi is distributed via the Git repo. Download it from the dist-firefox/ folder or clone/pull the repo to get the latest file.

Step 2

Open in Firefox

Drag the .xpi file into any Firefox window, or use File → Open File and select it.

Step 3

Confirm Install

Firefox will show a permissions prompt. Click Add to install. The extension persists across restarts.

Step 4

Pin & Verify

Click the extensions puzzle icon in the toolbar, find BambiBrowser+, and pin it. Open the popup to confirm the status view loads.

Post-Install Checks

Confirm the extension is running correctly before moving on.

Popup

Status View Loads

Clicking the icon should show the compact status popup — helper state, monitor mode, input lock, and playback section.

Settings

Cog Opens Full Tab

Clicking the cog icon in the popup should open a full-size settings tab.

Firefox 140 or newer is required. If you're on an older version, update Firefox before installing.

Updating the Extension

Firefox does not auto-update unlisted extensions. When a new version is released:

Option A

Install over existing

Get the new .xpi from the Git repo (dist-firefox/) and drag it into Firefox. It will replace the old version and keep your settings.

Option B

Remove then reinstall

Go to about:addons, remove BambiBrowser+, then install the new .xpi fresh.
